# projeto-demo-ia

Mock de automação de verificação de código e geração de documentação técnica.

## Stack
- n8n Community Edition (self-hosted)
- GitHub (controle de versão e webhooks)
- Claude (análise assistida)

## Estrutura
- `app/` — código da aplicação
- `docs/` — documentação gerada automaticamente
- `reports/` — relatórios JSON de cada execução
- `logs/` — registros de execução do pipeline
